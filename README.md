# Amrita Mazumdar | Luxury Makeup Artist 💄

A high-end, responsive portfolio website for **Amrita Mazumdar**, a luxury bridal, party, and event makeup artist based in Asansol, West Bengal.

![Project Preview](images/bridal1.jpeg)

## ✨ Features

- **Luxury Bento Gallery**: A visually stunning 9-item grid showcasing high-quality bridal transformations.
- **WhatsApp Integration**: Streamlined booking form that sends structured appointment details directly to the artist's WhatsApp.
- **Modern Design**: Clean, elegant aesthetics using premium typography (Playfair Display & Poppins) and a responsive layout.
- **SEO Optimized**: Pre-configured meta tags for search engines and social media previews (Open Graph).
- **Interactive Elements**: Smooth scrolling, hover effects, and a floating WhatsApp contact button.

## 🚀 Getting Started

This is a static website. You don't need any special environment to run it.

1. Clone or download this repository.
2. Ensure the directory structure is maintained:
   ```text
   /
   ├── webpage.html
   └── images/
       ├── bridal1.jpeg
       └── ...
   ```
3. Open `webpage.html` in any modern web browser (Chrome, Safari, Firefox, Edge).

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Layout (CSS Grid & Flexbox) and animations.
- **JavaScript**: Form handling and dynamic copyright dates.
- **Google Fonts**: Playfair Display for elegance, Poppins for readability.
